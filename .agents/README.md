# .agents Index

이 디렉터리는 이 리포지토리에서 작업할 때 먼저 참고해야 하는 문서의 기준점이다.

## 읽는 순서

1. `../AGENTS.md`
   - 프로젝트 전체 개요
   - 실행 흐름
   - 주요 디렉터리와 데이터 구조
2. `./AGENTS.md`
   - 프로젝트 도메인 설명
   - 구현된 구성요소 요약
3. `./rules/README.md`
   - 규칙 문서 인덱스
   - 어떤 상황에서 어떤 문서를 먼저 볼지 안내
4. 작업과 직접 관련된 규칙 문서
   - 에이전트/학습 로직이면 `./rules/agents.md`
   - 환경, 상태, 보상, dynamics면 `./rules/env.md`

## 사용 원칙

- 코드 전체를 먼저 훑지 말고 이 문서 트리부터 읽는다.
- 필요한 구현 파일만 문서 기준으로 좁혀서 확인한다.
- 문서와 코드가 다르면 코드를 기준으로 판단한다.
- 코드 기준으로 판단한 뒤 `.agents` 문서를 최신 상태로 유지한다.

## 문서 맵

- 루트 진입점: `../AGENTS.md`
- 프로젝트 개요: `./AGENTS.md`
- 규칙 인덱스: `./rules/README.md`
- RL 에이전트: `./rules/agents.md`
- 환경과 상태/보상: `./rules/env.md`
